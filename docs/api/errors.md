# Error Codes

## HTTP Status Codes

| Code | Meaning |
|------|---------|
| 200 | OK |
| 201 | Created |
| 400 | Bad Request |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 429 | Rate Limited |
| 500 | Server Error |

## Common Errors

- `PROJECT_NOT_FOUND` - Project doesn't exist
- `AUTH_REQUIRED` - Authentication needed
- `RATE_LIMITED` - Too many requests
